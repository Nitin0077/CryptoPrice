About CryptoPulse

CryptoPulse is a live dashboard that brings together real-time data for:

  • Top Cryptocurrencies by Market Cap
  • Indian Stock Market Gainers (via IndianStockAPI)
  • Popular Indian Mutual Funds (NAV, category)
  • Global Market Overview (BTC Dominance, Cap, Volume)
  • Top Crypto Exchanges and Liquidity Stats
  • Upcoming Indian IPOs

Tech Stack

Built using Angular 20, TypeScript, RxJS, CSS and REST APIs.

APIs Used

  • LiveCoinApi
  • IndianStockAPI
